`Type : 제목 #이슈번호`

`본문`

**⚙️ Type**

- `feat` : 새로운 기능에 대한 커밋
- `fix` : 수정에 대한 커밋
<<<<<<< HEAD
- `docs` : 문서 수정에 대한 커밋
- `styled` : 코드 스타일 혹은 포맷 등에 관한 커밋
- `set(setting)` : 프로젝트 세팅에 대한 커밋

=> 예시
-   commit -m "feat : slider 기능 추가"
-   commit -m "fix : nav css 변경"
-   commit -m "docs : Readme.md 수정"
-   commit -m "setting : 공통적으로 사용하는 css 설정"
